# Aurya Hub — Landing Page

Landing page profissional para a Aurya Hub, empresa de consultoria estratégica de negócios com foco em serviços fiscais e de marketing.

## Stack

- **Framework:** Next.js 15 (App Router)
- **Linguagem:** TypeScript
- **Estilos:** Tailwind CSS
- **Fontes:** Cinzel (títulos), Arial (corpo)

## Desenvolvimento

```bash
npm install
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Estrutura

- `src/app/` — Rotas e layout (App Router)
- `src/components/` — Header, Footer, AnimateOnScroll
- `src/components/sections/` — Seções da landing (Quem Somos, O Que Fazemos, etc.)
- `public/` — Assets estáticos (logo)

## Recursos

- Design responsivo (mobile-first)
- Navegação com smooth scroll
- Cores extraídas do logo (azul, marrom, roxo)
- Acessibilidade (WCAG AA): contraste, foco, skip link, ARIA
- SEO: meta tags, Open Graph, HTML semântico
- Animações suaves com respeito a `prefers-reduced-motion`
- CTA principal para WhatsApp
